# 基于 scikit-learn 的金融买方预测项目

## 项目概述
这个项目展示了如何使用 scikit-learn 在金融买方场景中进行简单的回归预测。我们将构建一个示例模型，用历史数据预测资产收益率，并在 Notebook 中可视化结果。

## 使用的工具与库
- **scikit-learn**：用于模型训练与预测。
- **pandas**：用于数据处理。
- **matplotlib**：用于结果可视化。

## 项目亮点
- **金融买方视角**：聚焦于买方机构常见的收益预测问题。
- **轻量级原型**：演示了如何用 scikit-learn 快速搭建一个简单、实用的回归模型。

## 如何运行
1. 在浏览器中打开 Jupyter Notebook 或 Google Colab。
2. 将本项目的代码粘贴到 Notebook 中执行。
3. 观察模型预测结果的可视化输出。

## 文件说明
- `main.ipynb`：包含了完整的 scikit-learn 模型构建与可视化代码。
