# 我的金融预测项目

## 项目概述
这个项目演示了如何使用 scikit-learn 来进行一个简单的金融回归预测。我们使用了随机森林回归模型，并将实际值和预测值进行了可视化对比。

## 使用方法
1. 在浏览器中打开 Jupyter Notebook 或 Google Colab。
2. 复制项目代码到 Notebook 中并运行。
3. 观察图表结果，了解模型的预测效果。

## 文件结构
- `main.ipynb`：主 Notebook 文件，内含模型代码和可视化。
